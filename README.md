## Requirements

- Java 17 or later
- Kotlin 2.4.0
- KSP 2.3.9 with KSP2 enabled
- Komapper 7.0.0
- JUnit 6.1.0

## Commands

```bash
./gradlew run
./gradlew build
./gradlew test
```
